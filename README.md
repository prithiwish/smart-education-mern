# Smart Education Platform

A full-stack MERN (MongoDB, Express, React, Node.js) based Smart Education platform designed to support modern digital learning with a scalable backend and an interactive frontend.

---

## Project Overview

The Smart Education Platform is a web-based application that aims to improve the digital learning experience by providing a structured system for managing educational content. The project follows standard full-stack architecture with a clear separation between frontend and backend.

---

## Features

- React-based frontend
- Node.js and Express backend
- Modular and scalable project structure
- MongoDB database integration
- Clean REST API design

## Installation & Setup

### Clone the repository

git clone https://github.com/your-username/smart-education-mern.git

### Frontend setup

cd frontend  
npm install  
npm start

### Backend setup

cd backend  
npm install  
npm run dev

---

## Environment Variables

Create a `.env` file inside the backend folder:

PORT=5000  
MONGO_URI=your_mongodb_connection_string
